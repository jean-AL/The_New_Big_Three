SERIE LIVE ONE PIECE

Cette série en prises de vues réelles raconte les aventures d'un pirate peu ordinaire d'après le célèbre manga d'Eiichiro Oda.

Le projet a commencé en 2017 suite à un message de ODA pour les fans

La série va être disponible sur NETFLIX a partir du 31 aout 2023, elle abordera la 1ere SAGA ( EAST BLUE )

Actuellement L’anime de ONE PIECE se trouve dans L’arc Onigashima ( Wano-Kuni ) avec 1068 épisode
